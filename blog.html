Important FAQ

What is Document Object Model (DOM)

DOM or Document Object Model is a model of HTML. DOM makes object of html elements so that Javascript can easily interact with html elements.

How do you select an element from DOM

We can select an element from DOM by using ID or Class Name or any other Query Selector

How would you manipulate the DOM using JavaScript?

Javascript manipulates the DOM by selecting elements by their id or class or tag name (using methods like getElementById, getElementsByClassName, getElementsByTagName, etc.) and change their internal content by using innerText or innerHTML.

What is DOM tree and DOM nodes?

DOM treats html tags as a tree structure. HTML tag is the root of the document. Inside HTML tag, there are body and head tag. They are the children of HTML tag. body may have lots of div and section. they are the children of body tag. Every elements in HTML structure treat that way. So everything is looks like a tree. Every html tag is a DOM node.

How is event propagation handled in the DOM?

In event Propagation, there are three phases:

1. Capturing 2. Target 3. Bubbling.
.,mnb vc In the capturing phase, the event work from top to bottom. Root node to target node. This is called Capturing phase. When event handle at the target node that is called Target phase and Finally, In bubbling phase, the event bubbles that means it will work from the target node to top of the node or root node. Event handler can trigger any of these three phases. But by default in DOM, it is trigger at bubbling phase. We can prevent default nature by applying stopPropagation() method. This method ensure the event handling at the target phase. Prevent the event handling from capturing and bubbling phase
