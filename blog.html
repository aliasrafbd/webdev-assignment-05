How would you manipulate the DOM using JavaScript?

JavaScript manipulates the DOM by selecting elements and changing their properties or contents. The ‘document’ object is used to access the HTML document, with methods like getElementById() or querySelector() for selection.

What is DOM tree and DOM nodes?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the structure of a document as a tree, known as the DOM tree. Each element in this tree is called a node. There are several types of nodes including Element Nodes, Text Nodes, and Attribute Nodes.

How is event propagation handled in the DOM?

Event propagation in the DOM is managed through three phases: capturing, target, and bubbling. In the capturing phase, the event descends from the root to the target node. The target phase involves the event being handled at the target node. Finally, during the bubbling phase, the event bubbles up from the target node back to the root. Event handlers can be registered for any of these phases, but by default, they are set for the bubbling phase. To stop propagation, the method stopPropagation() can be used. This prevents further propagation of the current event in the capturing and bubbling phases.
